## Overview
This repository showcases an interactive car sales dashboard created using Tableau to analyze sales performance for a car dealership.

## Project Focus
- Year-to-Date (YTD) and Year-over-Year (YOY) sales performance
- Average price and cars sold analysis
- Weekly sales trends
- Sales distribution by body style and color
- Dealer region and company-wise sales performance

## Key KPIs
- YTD Total Sales
- YOY Growth in Total Sales
- YTD Average Price
- YOY Growth in Average Price
- YTD Cars Sold
- YOY Growth in Cars Sold

## Deliverables
- Final dashboard screenshots
- Summary document with objectives and recommendations

## Tools Used
- Tableau
  - Calculated Fields
  - Interactive Filters
  - Parameters
  - Dashboard Actions

## Scope
This repository highlights final dashboards and insights.  
Detailed calculations and analysis logic are explained during interviews.
